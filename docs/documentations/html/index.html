Welcome to PyFOS’ Documentation!¶

PyFOS contains open source Python modules to facilitate mangement of FOS switches.

PyFOS Infrastructure Modules

  • pyfos_auth - PyFOS module to facilitate establishing a session to a FOS switch.
  • pyfos_rest_util - PyFOS module to provide REST support for FOS objects.
  • pyfos_util - PyFOS module to provide utility functions.

PyFOS FC Support Modules

  • pyfos_brocade_access_gateway - PyFOS Module to Provide REST Support for Access Gateway.
  • pyfos_brocade_fabric - PyFOS module to provide rest support for Fabric.
  • pyfos_brocade_fibrechannel - PyFOS module to provide rest support for FC ports.
  • pyfos_brocade_fibrechannel_diagnostics - PyFOS module to provide rest support for FC port diagnostics.
  • pyfos_brocade_fibrechannel_switch - PyFOS module to provide rest support for FC switch.
  • pyfos_brocade_zone - PyFOS module to provide rest support for Zoning.
  • pyfos_brocade_fdmi - PyFOS module to provide REST support for FDMI.
  • pyfos_brocade_fibrechannel_logical_switch - PyFOS module to provide REST support for Logical Switch objects.
  • pyfos_brocade_name_server - PyFOS module to provide rest support for Name Server.

PyFOS Extension Support Modules

  • pyfos_brocade_gigabitethernet - PyFOS module for GigE port objects.
  • pyfos_brocade_extension_ip_interface - PyFOS module for Extension IP.
  • pyfos_brocade_extension_ip_route - PyFOS module for Extension IP Route.
  • pyfos_brocade_extension_ipsec_policy - PyFOS module for IPSec Policy.
  • pyfos_brocade_extension_tunnel - PyFOS module for Extension Tunnel.
