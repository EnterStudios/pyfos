switchgigeportstateset - PyFOS util for setting GE port state.¶

The switchgigeportstateset provides for specific port op use case.

This module is a standalone script that can be used to set the switch GE port state on an extension platform.

  • inputs:
    • -L=<login>: Login ID. If not provided, interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, interactive

      prompt will request one.

    • -i=<IP address>: IP address

    • -n=<name>: GE port name expressed as slot/port.

    • -e=<enabled>: Enabled state of the GE port.

  • outputs:
    • Python dictionary content with RESTCONF response data.

ret = switchgigeportstateset.port_state_set(session, name, enabled)
