Object methods

get(session, name=None)¶

Returns a media_rdp object or a list of objects filled with attributes gathered from port media. If optional name is given, either an object matching the name of the port is returned or an empty object is returned with the corresponding error if no match is found.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:
  • session – session handler returned by pyfos_auth.login()
  • name – Interface slot port details in the form of interface/slot/port
Return type:

a media_rdp object if name is given or a list of objects if there are more than one
