cfgremove - PyFOS util for specific Zoning use case.¶

The cfgremove provides for specific Zoning use case.

This module is a standalone script and API that can be used to remove existing member(s) from existing cfg(s).

  • inputs:
    • -L=<login>: Login ID. If not provided, interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, interactive

      prompt will request one.

    • -i=<IP address>: IP address

    • -n=<cfg name>: string name of an existing cfg

    • -m=<member list>: list of members seperated by ”:”

    • -f=<VFID>: VFID or -1 if VF is disabled. If unspecified,

      VFID of 128 is assumed.

  • outputs:
    • Python dictionary content with RESTCONF response data
cfgremove.cfgremove(session, cfgs)¶

Remove from existing cfg(s) specified member(s)

Example usage of the method:

cfgs = [
            {
                "cfg-name": name,
                "member-zone": {"zone-name": members}
            }
          ]
result = cfgremove(session, cfgs)
Parameters:
  • session – session returned by login
  • cfgs – an array of cfg and members
Return type:

dictionary of return status matching rest response

use cases

  1. remove members from existing cfg
