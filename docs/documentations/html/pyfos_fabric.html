Object methods

static get(session)¶

Returns a fabric_switch object or a list of objects filled with attributes gathered throuth the session passed in. Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:a fabric_switch object if there is only one switch within fabric or a list of objects if there are more than one
