extensioncircuitdelete - PyFOS util for deleting a circuit object.¶

The extensioncircuitdelete provides circuit deletion functionality.

This module is a standalone script that can be used to delete an extension circuit.

  • inputs:
    • -L=<login>: Login ID. If not provided, interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, interactive

      prompt will request one.

    • -i=<IP address>: IP address

    • -n=<name>: slot/port name of the tunnel object for which this

      circuit is created.

    • -c=<circuit-id>: circuit Id of the circuit

  • outputs:
    • Python dictionary content with RESTCONF response data
