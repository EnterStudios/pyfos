Welcome to PyFOS’ documentation!¶

PyFOS contains open source Python modules to faciliatate mangement of FOS switches.

PyFOS infrastructure modules:

  • pyfos_auth - PyFOS module to faciliate estabishing a session to FOS switch
  • pyfos_rest_util - PyFOS module to provide rest support for FOS objects.
  • pyfos_util - PyFOS module to provide utility functions.

PyFOS FC support modules:

  • pyfos_fabric - PyFOS module to provide rest support for Fabric.
  • pyfos_switch - PyFOS module to provide rest support for FC switch.
  • pyfos_switchfcport - PyFOS module to provide rest support for FC ports.
  • pyfos_zone - PyFOS module to provide rest support for Zoning.

PyFOS extension support modules:

  • pyfos_brocade_gigabitethernet - PyFOS module to provide rest support for GigE objects.
  • pyfos_extension_circuit - PyFOS module to provide rest support for Extension Circuit objects.
  • pyfos_extension_ipaddress - PyFOS module to provide rest support for Extension IP Address objects.
  • pyfos_extension_iproute - PyFOS module to provide rest support for Extension IP Route objects.
  • pyfos_extension_ipsec_policy - PyFOS module to provide rest support for Extension IPSec Policy.
  • pyfos_extension_tunnel - PyFOS module to provide rest support for Extension Tunnel objects.
