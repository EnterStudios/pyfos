get(session, name=None)¶

Returns a fibrechannel_switch object or a list of objects filled with attributes gathered from switch. If optional name is given, either an object matching the WWN of the switch is returned or an empty object is returned.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:a fibrechannel_switch object if there is only one switch within fabric or a list of objects if there are more than one
patch(session)¶

Apply configurable attribute(s) within the object to switch.

Below is an example using individual sets:

# initialize the switch object
switch = pyfos_switch.fibrechannel_switch()
# set the enabled-state attribute to enable the switch
switch.set_enabled_state(pyfos_switch.ENABLE)
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)

Below is an example of combining object initialization and attribute sets:

# initialize the switch object and
# set the enable-state attribute
switch = pyfos_switch.fibrechannel_switch(
    {"enabled-state" : pyfos_switch.ENABLE})
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)
Parameters:session – session handler returned by pyfos_auth.login()
Return type:a dictionary of error or success information
