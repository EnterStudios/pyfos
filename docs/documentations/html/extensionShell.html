extensionShell - PyFOS util for emulating extension Shell.Â¶

The extensionShell provides CLI kind of shell functionality.

This module is a standalone script that can be used to basic operation using extension CLI. It does not cover all the CLI command supported.

  • Object supported:
    • ipif
    • iproute
    • fciptunnel
    • fcipcircuit
  • Operations:
    • portcfg
    • portshow

Example:

portshow ipif
portshow iproute
portshow fciptunnel
portshow fcipcircuit
