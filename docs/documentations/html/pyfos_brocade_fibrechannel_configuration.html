Object methods

get(session)¶

Returns a switch object with attributes from switch.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:dictionary of error or a switch object
patch(session)¶

Apply configurable attribute(s) within the object to switch.

Below is an example using individual sets:

# initialize the switch object
obj =
pyfos_brocade_fibrechannel_configuration.f_port_login_settings()
# set the enforce-login attribute to enable the
# login type precedence on the switch
obj.set_enforce_login(1)
# execute HTTP patch command to apply the object to the
# switch connected through session
obj.patch(session)

Below is an example of combining object initialization and attribute sets:

# initialize the switch object and
# set the max-logins attribute
obj =
pyfos_brocade_fibrechannel_configuration.f_port_login_settings(
    {"max-logins" : 1})
# execute HTTP patch command to apply the object to the
# switch connected through session
obj.patch(session)
